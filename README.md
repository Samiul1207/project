# E-Commerce Project

A full-featured Django-based online shopping website.

## Features
- User Registration & Login
- Product Catalog
- Shopping Cart
- Order Management
- Payment System (under development)

## Tech Stack
- **Backend**: Django
- **Database**: SQLite (can be changed to PostgreSQL)
- **Frontend**: HTML, CSS, Bootstrap
